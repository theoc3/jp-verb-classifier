# jp-verb-classifier

## Description
 Classify Japanese verbs by their tense/voice i.e. passive, past, participle, etc. 

## Model
 In progress

## Usage
### Command/Result
```
 python3 classify.py
```

 Currently prints out 4 examples: 
 ```
 ('聞いて', 'participle')
 ('食べられます', 'potential-polite')
 ('食べられる', 'potential')
 ('作りました', 'past-polite')
```
